# 모두의 전기차 충전소
전기차충전소를 쉽게 검색하고 유저가 직접참여해서 디테일한 정보를 추가할수있다.
만든이유: 주차장까지는 검색해서갔지만 몇층에 어디에있는지는 주차요원에게 항상 물어봐야하는 불편함
